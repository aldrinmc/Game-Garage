About the Site

This website stores informations of a game like the basic overview of the game and its genres. It will also show some screenshots and a trailer video of the game. In the page of the game users can click on a link that will redirect them to the game's website. Users can search games on the website using the search bar and alternatively use the category menu to narrow down the games into genres, logged in users will have the right to request games using the request page. Users not logged in will have lesser rights such as not being able write reviews and request games.
